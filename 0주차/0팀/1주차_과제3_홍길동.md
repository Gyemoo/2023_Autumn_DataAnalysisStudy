### 문제
변수 x,y,z를 선언하여 그 값을 각각 입력받고,
포인터를 이용하여 x값은 y에, y값은 z에, z값은 x에 바꿔넣고
마찬가지로 포인터로 각각의 값을 출력하는 코드를 작성하라

### 해답

```c
#include <stdio.h>

int main() {

	int x, y, z;
	int* px, * py, * pz, * tmp;

	px = &x;
	py = &y;
	pz = &z;

	scanf("%d %d %d", &x, &y, &z);

	tmp = pz;
	pz = py;
	py = px;
	px = tmp;

	printf("%d %d %d", *px, *py, *pz);

	return 0;
}
```
